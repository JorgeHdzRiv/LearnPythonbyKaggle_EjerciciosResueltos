# Exercises of course Learn Python by Kaggle

Este repositorio cuenta con los ejercicios resueltos de Learn Python por Kaggle el contenido que se muestra es de manera didactica
y los cuadernos contenidos son creados por Kaggle y modificados por mi para cumplir con los ejercicios que se pide.

Ampliamente te recomiendo visitar el curso donde se da la teoria y bases para poder realizar los ejercicios planteados.

**Los modulos learn tools funcionan solamente dentro de Kaggle pero sin embargo se puede dar a entender los contenidos y lo que se requiere en cada actividad**

**Por eso es recomendable acceder directamente a Kaggle para realizar el curso y probar tus habilidades**

**Los contenidos en la carpeta notebooks_guide son parte de los cuadernos de aprendizaje este contenido fue elaborado exclusivamente por el autor y solo estan en este repositorio para fines didacticos**
